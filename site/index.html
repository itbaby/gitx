Everything You Need to
Understand Code Changes

GitX brings AI intelligence directly into your Git workflow.

Branch Comparison

Side-by-side visual diff of any two branches with syntax highlighting and line-level change tracking.

AI-Powered Analysis

Ask questions about your code changes in natural language. Get instant, context-aware explanations and insights.

Commit History

Browse commit history across branches with detailed metadata, timestamps, and author information.

File History

Track changes to specific files over time. See what changed, when, and by whom across any time range.

Natural Language

Use plain language to query your repository. “Show me the last 5 commits” just works.

Self-Hosted

Runs entirely on your machine. Your code never leaves your network. Compatible with any OpenAI API endpoint.

How It Works

1

Open Your Repo

Point GitX to any local Git repository. No setup, no indexing — it reads your .git directory directly.

2

Pick Branches

Select any two branches to compare. GitX shows a rich diff view with syntax highlighting and stats.

3

Ask AI

Chat with the AI assistant about your changes. It has full access to your repo data and answers in context.

Built With Modern Tools

Go

Backend

Go + Gin for high-performance HTTP serving with go-git for native repository access.

Vue

Frontend

Vue 3 with TypeScript, Vite for fast HMR, and diff2html for beautiful diff rendering.

AI

AI Engine

OpenAI-compatible API with function calling. Works with GPT-4, GLM, or any compatible endpoint.

SSE

Streaming

Real-time SSE streaming with chunk buffering for smooth, responsive AI responses.

Get Started in 60 Seconds

Backend
cd backend
cp .env.example .env   # Add your API key
go mod tidy
go run cmd/main.go      # Starts on :8080
Frontend
cd frontend
npm install
npm run dev             # Starts on :5173

Requires Go 1.21+, Node.js 18+, and an OpenAI-compatible API key.
